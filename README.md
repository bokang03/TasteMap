# Taste Map
- 맛집 조회 및 저장 서비스
- 네이버 지도 API를 활용한 위치 기반 맛집 검색 및 저장(북마크) 서비스 

## 기술 스택
- Java 21
- Spring Boot 3.x
- Gradle
- Spring Web, Spring Security
- OpenAPI (Swagger UI)

### 주요 기능
- [x] 맛집 검색
    - [x] 네이버 API 연동
        - [x] 검색된 음식점 정보 조회
        - [x] 검색된 음식점 위치 정보 조회
        - [x] 검색된 음식점 이미지 조회
    - [x] 키워드 검색: 사용자가 입력한 키워드로 맛집 검색
- [x] 맛집 저장: 마음에 드는 맛집을 즐겨찾기에 추가하여 나중에 쉽게 접근
- [x] 임시 DB 모듈 구현
  - [x] 맛집 저장 
  - [x] 맛집 삭제
  - [x] 저장된 맛집 전체 조회 API
- [x] Swagger UI 연동

### 예외 처리
- [x] 음식점 이미지 없을 경우 기본 이미지 제공
- [x] 검색 결과가 없을 경우 예외 처리
- [x] 이미 저장된 맛집을 다시 저장하려고 할 때 예외 처리
- [x] 같은 음식점이 검색 결과에 여러 번 나타나는 경우 중복 제거


### 프로그래밍 요구 사항 1
- [ ]  indent(인덴트, 들여쓰기) depth를 3이 넘지 않도록 구현한다. 2까지만 허용
- [ ]  3항 연산자 사용 금지
- [ ]  함수(또는 메서드)가 한 가지 일만 하도록 최대한 작게 구현
- [ ]  JUnit 5와 AssertJ를 이용하여 정리한 기능 목록이 정상적으로 작동하는지 테스트 코드로 확인

### 프로그래밍 요구 사항 2
- [ ]  함수(또는 메서드)의 길이가 15라인을 넘어가지 않도록 구현
- [ ]  else 예약어를 쓰지 않도록 구현
- [ ]  java Enum을 적용하여 프로그램 구현
- [ ]  구현한 기능에 대한 단위 테스트를 작성한다. 단, UI(System.out, [System.in](http://system.in/), Scanner) 로직은 제외
