# Taste Map
- 맛집 조회 및 저장 서비스
- 네이버 지도 API를 활용한 위치 기반 맛집 정보 제공

### 주요 기능
- [ ] 맛집 검색
    - [ ] 네이버 API 연동:
        - [x] 검색된 음식점 정보 조회
        - [x] 검색된 음식점 위치 정보 조회
        - [ ] 검색된 음식점 이미지 조회
    - [ ] 키워드 검색: 사용자가 입력한 키워드로 맛집 검색
    - [ ] 위치 기반 검색: 사용자의 현재 위치를 기반으로 주변 맛집
- [ ] 맛집 저장: 마음에 드는 맛집을 즐겨찾기에 추가하여 나중에 쉽게 접근

### 예외 처리
- [ ] 이미 저장된 맛집을 다시 저장하려고 할 때 예외 처리
- [ ] 검색한 맛집이 없을 때 예외 처리
- [ ] 네이버 API 호출 실패 시 예외 처리
